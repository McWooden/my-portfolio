# Workspace Rules for McWooden/my-portfolio

## Chrome DevTools MCP Setup
- **Manual Launch Required**: The Chrome browser does not automatically start when opening this workspace or starting the MCP server.
- **Before using Chrome DevTools MCP tools** (e.g. navigation, screenshot, scripting):
  - Check if the debugging browser is running on port `9223`.
  - If it is not running, remind the user to run `chrome-debug` in their terminal to launch it.
